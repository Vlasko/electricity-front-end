Electricity Front-End Homepage

This project displays data collected from a TP Link smart plug and the Octopus Energy API.

It uses the following project to collect, store and upload electricity demand and price data on a
Raspberry Pi from WiFi enabled TP Link smart plug to AWS S3;

electricity-upload

This data is then downloaded from S3 onto a computer for processing and visualistion through this project;

electricity-processing

Finally, this front-end application is built using Django, hosted in this repository;

electricity-front-end
